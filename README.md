# Port Scanner
**Port Scanner** es otro escaner de puertos mas, permite diferentes funciones
El programa escanea la ip deseada con delay si se define, prueba antes de escanear el ping a no ser que el usuario lo desactive, guarda toda la info del escaneo en un archivo llamado scan.txt, escanea desde el puerto 0 hasta el 65535 en segundos, detecta que clase de servicios es cada puerto(ssh, http, ftp, tcp...), tiene una interfaz de consola con colores verdes y rojos para decir si hay errores o todo salio bien.
